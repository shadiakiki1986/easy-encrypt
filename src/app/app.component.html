This is my friend JP

JP wants to send me his phone number, but he doesn't trust social media.

Using this app, he can encrypt his phone number using my public key and send it to me over any social media text messenger.

The encrypted phone number can only be decrypted using my private key.

Decrypt using openssl in terminal 
  # Save to file
  cat > encrypted.txt
  # paste encrypted message
  # Ctrl+C

  # decode base64
  base64 -d encrypted.txt > encrypted.bin

  # decrypt using PEM private key
  openssl rsautl -decrypt -in encrypted.bin -inkey ~/.ssh/id_rsa.pem
